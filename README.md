# Jersey REST API CRUD Application

This project is a beginner-friendly implementation of RESTful Web Services using **Jersey (JAX-RS)**. It demonstrates how to build and test REST APIs with **Java**, **JDBC**, **MySQL**, and **Postman**.

## Features

* REST API development using Jersey (JAX-RS)
* MySQL database integration using JDBC
* Complete CRUD operations:

  * GET – Retrieve all records and a specific record by ID
  * POST – Create new records
  * PUT – Update existing records (or create if the record does not exist)
  * DELETE – Remove records by ID
* JSON and XML request/response support
* Parameterized SQL queries using PreparedStatement
* Tested using Postman

## Technologies Used

* Java
* Jersey (JAX-RS)
* Maven
* JDBC
* MySQL
* Apache Tomcat
* Postman

## Learning Outcomes

This project helped in understanding:

* REST API architecture
* HTTP methods (GET, POST, PUT, DELETE)
* Resource classes and JAX-RS annotations
* JSON/XML serialization and deserialization
* JDBC connectivity and CRUD operations
* Integration of REST APIs with a relational database
